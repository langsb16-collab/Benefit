# 지역화폐 O2O 플랫폼 Benefit

## 프로젝트 개요

**목표**: 지역 소상공인의 디지털 전환을 지원하고, 지역화폐와 연계하여 골목 상권의 자생력을 강화하며, 새로운 지역 일자리를 창출하는 공공·민관협력 O2O 플랫폼

## 🗺️ 완전한 사이트맵 구조

### 1. 홈 (Home)
- 공지/긴급 안내
- 지역화폐 캐시백 배너 (5~10%)
- 오늘의 인기 상점
- 지역 행사/축제 홍보
- 전통시장 장보기 추천

### 2. 주변가게·배달 (Nearby Stores & Delivery)
- **음식 배달**: 배달/포장/예약 주문
- **포장 주문**: 픽업 서비스
- **예약 주문**: 사전 예약
- **카테고리별 상점**: 동네 반찬/카페/마트/베이커리

**주요 기능**:
- 지역화폐 가맹점 우선 노출
- 할인·캐시백 적용 상점
- 거리순/인기순 정렬
- 실시간 메뉴/리뷰

### 3. 장보기·전통시장 (Traditional Market)
- **전통시장 장보기**: 재래시장 카테고리별
- **합배송**: 묶음배송 시스템
- **특산물·로컬푸드**: 지역 농산물 직거래
- **시장 쿠폰**: 지자체 발행 상생쿠폰

### 4. 공동구매 (Group Buying)
- **진행중 공동구매**: 실시간 참여율 표시
- **마감된 공동구매**: 완료된 공동구매 내역

### 5. 지역행사·축제 (Local Events & Festivals)
- **행사 목록**: 지역 축제 정보
- **행사 티켓**: O2O 티켓 구매
- **스탬프 투어**: GPS 기반 스탬프 투어

### 6. 지역화폐 결제 (Local Currency Payment)
- **지역화폐 등록**: 모바일형/카드형 등록
- **잔액조회**: 실시간 잔액 확인
- **결제내역**: 거래 이력 조회
- **정산내역**: 캐시백/정산 내역

**핵심 기능**:
- 즉시 할인·캐시백 (최대 10%)
- 복합 결제 (잔액 부족 시 자동 카드 결제)
- 자동 환불·실패 복원

### 7. 마이페이지 (My Page)
- **주문내역**: 주문/환불 내역
- **쿠폰함**: 보유 쿠폰 관리
- **지역화폐 잔액**: 실시간 잔액
- **즐겨찾기**: 즐겨찾는 상점
- **고객센터**: 문의/FAQ

### 8. 소상공인센터 (Merchant Center)
- **상점 정보 관리**: 사업자 인증, 영업시간 설정
- **메뉴 관리 (OCR)**: 사진·가격 자동 인식
- **POS 연동**: 자동 주문 수신
- **주문·배차 관리**: 실시간 주문 접수
- **매출·통계**: 요일별/시간대별 분석
- **정산관리**: 수수료 1~2%, 익일 정산

### 9. 지자체관리자 (Admin Dashboard)
- **정책 대시보드**: 지역경제 지표 시각화
- **데이터 분석**: 상권 활성화 지표
- **공공 쿠폰 발행**: 상생쿠폰 시스템
- **설문·공지**: 긴급공지/정책 설문
- **가맹점 관리**: 가맹점 승인/관리
- **배달대행 관리**: GPS 실시간 매핑

## 🎨 기술 스택

- **Backend**: Hono.js (TypeScript)
- **Frontend**: Vanilla JavaScript
- **스타일**: Tailwind CSS + Custom CSS
- **아이콘**: Font Awesome 6
- **배포**: Cloudflare Pages
- **개발 서버**: PM2 + Wrangler

## 🚀 빠른 시작

### 1. 의존성 설치
```bash
cd /home/user/webapp
npm install
```

### 2. 빌드
```bash
npm run build
```

### 3. 개발 서버 시작
```bash
# 포트 정리
npm run clean-port

# PM2로 서버 시작
pm2 start ecosystem.config.cjs

# 서버 테스트
curl http://localhost:3000

# 로그 확인
pm2 logs webapp --nostream
```

## 📡 API 엔드포인트

### GET /api/menu
메뉴 구조 데이터를 반환합니다.

**Response**:
```json
{
  "mainMenu": [
    {
      "id": "home",
      "name": "홈",
      "icon": "fa-home",
      "path": "/",
      "subMenu": [...]
    }
  ]
}
```

### GET /api/faq?lang={언어코드}
FAQ 데이터를 반환합니다.

**Query Parameters**:
- `lang`: 언어 코드 (ko, en, zh, ja, vi, th, ar)
